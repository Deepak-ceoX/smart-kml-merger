# Smart KML Studio

Open-source, offline KML/KMZ merger and optimizer.

## Features

* Merge multiple KML files
* Remove duplicates
* Optimize folder structure
* Export KML/KMZ
* Local map preview
* No API keys
* No cloud upload
* Works fully offline

## Quick Start

git clone https://github.com/Deepak-ceoX/smart-kml-studio.git
cd smart-kml-studio
npm install
npm run dev

Open http://localhost:3000

## Privacy

All files are processed locally in the browser. No data is uploaded to any server.
